# `hy-store`

## `Usage`

### `install`

```shell
yarn add hy-store
```

### `Example:`

```js
import { Menu, AutoComplete } from 'hy-store';
import 'hy-store/lib/index.css';

ReactDOM.render(
  <>
    <Menu>
      <Menu.Item>第一个Item</Menu.Item>
    </Menu>
    <AutoComplete fetchSuggestion={() => []} />
    document.getElementById('root')
  </>
);
```
